📝 User Registration Form

A simple and responsive user registration form built using basic HTML. This project is ideal for beginners learning form handling and structure in web development.

🚀 Features
Clean and minimal UI
Input fields for user details
Basic HTML form validation
Easy to customize and extend
📂 Project Structure
├── index.html
└── README.md
📄 Description

This project contains a basic user registration form that collects essential user information such as:

Full Name
Email Address
Password
Confirm Password

It demonstrates how to structure a form using HTML and apply simple validation using built-in attributes.

🛠️ Technologies Used
HTML5
▶️ How to Run

Download or clone this repository:

git clone https://github.com/your-username/user-registration-form.git

Navigate to the project folder:

cd user-registration-form
Open index.html in your browser:
Double-click the file
OR
Right-click → Open with browser
